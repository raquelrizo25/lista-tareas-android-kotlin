# 📱 App Lista de Tareas (Kotlin - Android)

Aplicación móvil desarrollada en Kotlin para la gestión de tareas, con una interfaz sencilla y funcional.

## 🚀 Funcionalidades

- Añadir nuevas tareas
- Visualizar lista de tareas
- Interfaz básica en Android (XML)
- Gestión sencilla de datos en memoria

## 🛠️ Tecnologías utilizadas

- Kotlin
- Android Studio
- XML (layouts)
